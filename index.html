BibleProject is a nonprofit animation studio and Bible-education platform creating free resources to help people experience the Bible as a unified story leading to Jesus. This site hosts open source tools, fonts, and assets that developers and designers can use in projects that share or reference BibleProject content.

📦 Repositories Browse public GitHub repositories — fonts, design tokens, icon libraries, and more. ⬇️ Downloads Download brand assets directly: font packages, logo files, and other distributable resources. 📖 Documentation Guides for getting started, integrating fonts, and contributing to BibleProject open source projects.
